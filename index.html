Did You Know?

🌐

The first website ever made is still online

Tim Berners-Lee published the first website on August 6, 1991 at CERN. It's still accessible at info.cern.ch!

🎨

CSS was first proposed in 1994

Håkon Wium Lie proposed CSS at CERN. It took 2 years before the first browser (IE 3) supported it. Before that, styling was purely presentational HTML.

⚡

JavaScript was created in 10 days

Brendan Eich created the first version of JavaScript in just 10 days in May 1995 for Netscape Navigator. Originally called Mocha, then LiveScript.

🐞

The first computer bug was a real bug

In 1947, Grace Hopper found a moth stuck in the Harvard Mark II computer relay — coining the term "debugging". The moth is preserved at the Smithsonian.
